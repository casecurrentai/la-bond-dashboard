## Bond Amount: NOT in roster table
- Clicking an inmate name redirects to VINELink (vinelink.com/person-detail/offender/{id})
- VINELink detail page shows: DOB (partial), ID Number (partial), Custody Status, Custody Detail, Book Date, Location, Reporting Agency
- NO bond amount on VINELink detail page
- Bond amounts are NOT available through LA VINE / VINELink

## Conclusion
LA VINE provides: name, DOB, race, gender, arrest date, custody status, facility
LA VINE does NOT provide: bond amount, charges, booking number

## Strategy
- Use LA VINE for inmate PRESENCE verification (is this person currently incarcerated?)
- For bond amount: must use the no-bond fallback workflow (call booking desk)
- OR: check if parish has a separate bond lookup portal
